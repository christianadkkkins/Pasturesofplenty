# Pasturesofplenty

Pasturesofplenty is a compact public snapshot of the current cross-domain geometric early-warning work in this project.

The repo is organized around three frozen experiment tracks:

- cardiac ST-event analysis on PhysioNet LTST
- solar-wind / geomagnetic storm analysis on OMNI HRO2 one-minute data
- micro-PMU event analysis on the LBNL October 1 dataset

Start here:

- [RESULTS.md](RESULTS.md): high-level experiment index
- [DATA_ACCESS.md](DATA_ACCESS.md): exact dataset names, sources, and local access paths
- [paper/README.md](paper/README.md): current preprint-facing text
- [results/README.md](results/README.md): reproducibility package and runner notes

The main frozen summary table is [results/RESULTS_SUMMARY.csv](results/RESULTS_SUMMARY.csv).
